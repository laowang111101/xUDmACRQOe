# 项目简介 python917

本项目是一个基于Python开发的日常管理系统，旨在提高公司内部工作效率和协调各部门之间的工作流程。

## 技术栈

- **前端**：HTML、CSS、JavaScript、Layui
- **后端**：Django
- **语言**：Python
- **数据库**：MySQL

## 功能模块

系统主要包含以下功能模块：

1. 用户管理
   - 用户注册
   - 用户登录
   - 用户权限设置
2. 部门管理
   - 部门信息维护
   - 员工信息管理
3. 任务管理
   - 任务创建与分配
   - 任务进度跟踪
   - 任务报告
4. 文档管理
   - 文件上传与下载
   - 文件分类与检索
   - 版本控制

## 系统角色

- 系统管理员：负责系统整体维护和用户权限管理
- 普通用户：执行日常任务，使用系统功能

## 运行环境

- 操作系统：Linux、Windows
- Python版本：3.6+
- 数据库：MySQL 5.7+

## 部署步骤

1. 准备Python运行环境
2. 安装依赖库
3. 配置数据库
4. 运行后端服务
